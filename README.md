# 광고 정산 자동화 시스템

광고 대시보드 스크린샷을 OCR로 분석하여 자동으로 정산서 엑셀 파일을 생성합니다.

## 📋 주요 기능

- **OCR 기반 데이터 추출**: Google Cloud Vision API를 사용한 정확한 텍스트 인식
- **다중 매체 지원**: 구글 광고, 네이버 쇼핑검색, 애플서치애즈(ASA)
- **자동 정산 계산**: 매체별 마크업, VAT 자동 계산
- **월별 관리**: 월별 폴더 구조로 정산 데이터 체계적 관리
- **엑셀 자동 생성**: 매체별 시트 + 요약 시트 + 스크린샷 첨부

## 🚀 사용 방법

### 1. 월별 폴더 생성 및 스크린샷 준비

```
data/screenshots/
├── 2025-11/
│   ├── google.png            # 구글 광고 대시보드
│   ├── naver_shopping_1.png  # 네이버 쇼핑검색 계정 1
│   ├── naver_shopping_2.png  # 네이버 쇼핑검색 계정 2
│   └── asa.png               # 애플서치애즈
├── 2025-12/
│   ├── google.png
│   └── ...
```

**파일명 규칙:**
- 구글: `google.png` 또는 `google_*.png`
- 네이버: `naver*.png` (여러 파일 가능, 계정별로)
- ASA: `asa.png` 또는 `asa_*.png`

### 2. 정산 실행

**이번 달 자동 처리:**
```bash
python run_settlement.py
```

**특정 월 처리:**
```bash
python run_settlement.py 2025-11
```

### 3. 결과 확인

생성된 파일 위치: `output/settlement_YYYY-MM_YYYYMMDD_HHMMSS.xlsx`

## 📊 정산 규칙

### 구글 광고
- 15% 마크업 적용
- 원본 비용 → 마크업 포함 비용(VAT-) → VAT 포함 비용(VAT+)

### 네이버 쇼핑검색
- 대시보드 비용은 VAT 포함
- VAT 제외 금액 계산 (비용 ÷ 1.1)

### 애플서치애즈 (ASA)
- USD → KRW 환율 적용 (기본: 1464.8)
- 15% 마크업 적용

## 📝 출력 엑셀 형식

|---------|------|--------|------|---------------------|-------------------|

| 네이버 | 5,000,000 | 5,500,000 |
| **전체 합계** | **15,000,000** | **16,500,000** |

## ⚠️ 주의사항

1. **스크린샷 품질**: 고해상도 스크린샷 권장 (텍스트가 선명해야 함)
2. **대시보드 설정**: 
   - 기간: 매월 1일 ~ 말일
   - 모든 캠페인 표시되도록 설정
3. **파일명**: 규칙에 맞게 정확히 명명
4. **네이버 계정**: 여러 계정이 있으면 각각 별도 파일로 저장

## 🔧 문제 해결

### OCR 정확도가 낮을 때
- 스크린샷 해상도 높이기
- 불필요한 UI 요소 제거
- 표 전체가 보이도록 캡처

### 비용이 정확하지 않을 때
- 대시보드 표시 형식 확인
- OCR 결과 로그 확인 (`data/ocr_results/`)
- 파싱 규칙 조정 필요 시 문의
